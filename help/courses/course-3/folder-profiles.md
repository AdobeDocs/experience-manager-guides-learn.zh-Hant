# 資料夾配置檔案

提供AEM對配置工具的快速訪問。 通過自定義資料夾配置檔案，不同部門或產品可以具有獨特的模板、創作環境、條件屬性配置檔案、代碼段，甚至Web Editor配置。

檔案中提供了您可能選擇用於本課的示例檔案 [資料夾配置檔案.zip](assets/folderprofiles.zip)。

>[!VIDEO](https://video.tv.adobe.com/v/342758)

## 訪問資料夾配置檔案

通過「資料夾配置檔案」表徵圖管理配置。

1. 在導航螢幕中，按一下 [!UICONTROL **工具**] 表徵圖

   ![工具表徵圖](images/reuse/tools-icon.png)

2. 選擇 **參考線** 的下界。

3. 按一下 [!UICONTROL **資料夾配置檔案**] 平鋪。

   ![資料夾配置檔案](images/reuse/folder-profiles-tile.png)

4. 選擇所需的配置檔案。 例如，選擇 **全局配置檔案**，這是預設配置檔案。

   ![全局配置檔案](images/lesson-3/global-profile-tile.png)

## 編輯全局配置檔案中的條件屬性

訪問「全局配置檔案」後，可以編輯其配置。 除非另有指定，否則全局配置檔案設定將應用於所有用戶。

1. 在全局配置檔案中，選擇 **條件屬性** 頁籤。

2. 按一下 [!UICONTROL **編輯**] 的下界。

   ![條件屬性](images/lesson-3/edit-conditional-attributes.png)

3. 按一下&#x200B;[!UICONTROL **「新增」**]。

4. 填充 **名稱**。 **值**, **標籤** 的子菜單。

   ![新條件](images/lesson-3/new-condition.png)

5. 按一下 [!UICONTROL **保存**] 左上角。
新條件現在可供所有用戶使用。 可以在「內容屬性」面板中選擇它，並根據需要將其應用於內容。

## 生成新資料夾配置檔案

除了預設的全局配置檔案外，您還可以建立自己的自定義配置檔案。

1. 在導航螢幕中，按一下 [!UICONTROL **工具**] 表徵圖

   ![工具表徵圖](images/reuse/tools-icon.png)

2. 選擇 **參考線** 的下界。

3. 按一下 [!UICONTROL **資料夾配置檔案**] 平鋪。

   ![資料夾配置檔案](images/reuse/folder-profiles-tile.png)

4. 按一下&#x200B;[!UICONTROL **建立**]。

5. 在「建立資料夾配置檔案」對話框中。

   a.命名配置檔案。

   b.指定路徑。

   c.按一下 [!UICONTROL **建立**]。

   ![建立資料夾配置檔案](images/lesson-3/create-folder-profile.png)

在「資料夾配置檔案」頁面上顯示具有新配置檔案名稱的磁貼。

## 從「常規」頁籤添加管理用戶

管理用戶有權更新資料夾配置檔案的條件屬性、創作模板和輸出預設。

1. 按一下磁貼以開啟所需的資料夾配置檔案。

   ![編輯資料夾配置檔案](images/lesson-3/edit-folder-profile.png)

2. 選擇 **常規** 頁籤。

3. 按一下 [!UICONTROL **編輯**] 左上角。

4. 在管理員用戶下，從下拉清單中選擇用戶或鍵入用戶名。

5. 按一下&#x200B;[!UICONTROL **「新增」**]。

   如果需要，可以添加多個管理員用戶。

   ![添加管理員](images/lesson-3/add-admin.png)

6. 按一下 [!UICONTROL **保存**] 在螢幕上的右上角。

管理用戶現在已分配到此配置檔案。

## 從「條件屬性」頁籤添加新訪問群體

訪問「全局配置檔案」後，可以編輯其配置。 除非另有指定，否則全局配置檔案設定將應用於所有用戶。

1. 在所需的資料夾配置檔案中，選擇 **條件屬性** 頁籤。

2. 按一下 [!UICONTROL **編輯**] 的下界。

   ![編輯條件屬性2](images/lesson-3/edit-conditional-attributes-2.png)

3. 按一下&#x200B;[!UICONTROL **「新增」**]。

4. 填充 **名稱**。 **值**, **標籤** 的子菜單。

   按一下 [!UICONTROL **加**] sign允許您為命名屬性添加其他值和標籤對。

   ![添加條件](images/lesson-3/add-conditions.png)

5. 按一下 [!UICONTROL **保存**] 左上角。

新條件屬性已添加到此配置檔案。

## 從「創作模板」頁籤中選擇模板和映射

指AEM南提供現成的創作模板和地圖。 您可以將其限制為特定作者。 預設情況下，模板儲存在DITA模板資料夾中的「資產」位置。

1. 在所需的「資料夾配置檔案」中，選擇「創作模板」頁籤。

2. 按一下螢幕左上角的「編輯」。

3. 添加映射模板。

   a.從 **映射模板** 下拉清單，從可用映射中選擇一個選項。

   b.按一下 [!UICONTROL **添加**]。

   ![映射模板](images/lesson-3/map-templates.png)

4. 添加主題模板。

   a.從 **主題模板** 下拉清單，從可用模板中選擇一個選項。

   ![主題模板](images/lesson-3/topic-templates.png)

5. 按一下&#x200B;[!UICONTROL **「新增」**]。

6. 根據需要添加其他主題模板。

7. 完成後，按一下 [!UICONTROL **保存**] 左上角。

新的創作模板已添加到此配置檔案。

## 從「輸出預設」頁籤中刪除非基本預設

您可以根據資料夾配置檔案配置每個輸出預設。 不需要的輸出預設應刪除。

1. 在所需的資料夾配置檔案中，選擇 **輸出預設** 頁籤。

2. 在左面板上，選中不需要的任何預設的複選框。

   ![刪除預設](images/lesson-3/delete-presets.png)

3. 按一下 [!UICONTROL **刪除預設**] 的下界。

4. 在「刪除預設」對話框中，按一下 [!UICONTROL **刪除**]。

   ![刪除](images/lesson-3/delete.png)

現在，顯示的唯一輸出預設是將要使用的預設。

## 從「XML編輯器配置」頁籤上載代碼段

1. 在所需的資料夾配置檔案中，選擇 **XML編輯器配置** 頁籤。

2. 在XML編輯器代碼段下，按一下 [!UICONTROL **上載**]。

   ![上傳代碼片段](images/lesson-3/upload-snippet.png)

3. 導航到先前建立的代碼段。

4. 按一下 [!UICONTROL **開啟**]。

5. 按一下 [!UICONTROL **保存**] 左上角。

您已成功修改編輯器配置以包括代碼段。

## 在儲存庫中指定資料夾配置檔案

在編輯器中，可以看到對資料夾配置檔案所做的修改的結果。

1. 導航到 **儲存庫視圖**。

2. 按一下要處理的內容的資料夾。

3. 按一下 [!UICONTROL **用戶首選項**] 表徵圖

   ![使用者偏好設定](images/lesson-3/hr-user-prefs.png)

4. 在「用戶首選項」對話框中，從下拉清單中選擇所需的資料夾配置檔案。

   ![選擇用戶首選項](images/lesson-3/select-user-pref.png)

5. 按一下「[!UICONTROL **儲存**]」。

您已將資料夾配置檔案應用於您的內容。 現在，在建立新DITA主題時，您將看到基於資料夾屬性的主題類型的受限清單。 「受眾條件」包含「全局」設定以及特定於「資料夾配置檔案」的設定。 您上傳的代碼段檔案建立了一組預設的代碼段供選擇。 「映射面板」顯示受限制的輸出預設。
