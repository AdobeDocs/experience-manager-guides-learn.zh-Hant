# 簡單內容建立工作流

「參考AEM線編輯器」具有多個快捷方式，可簡化內容建立工作流。 這些快捷方式允許用戶快速添加和修改影像、一次處理多個主題、更正錯誤、下載主題PDF以及使用版本和標籤。

>[!VIDEO](https://video.tv.adobe.com/v/342770)

## 新增影像

可以直接從本地驅動器添加映像。

1. 將影像直接拖放到主題中。 的 **上載資產** 對話框。

   ![「上載資產」對話框](images/lesson-15/upload-assets-dialog.png)

2. 修改到所需影像位置的資料夾路徑。

3. 將影像名稱更改為代表其目的的名稱。

4. 按一下 [!UICONTROL **上載**]。

## 修改影像

1. 通過拖放角來調整影像大小。

2. 通過拖放影像將影像移動到主題中的其他位置。

3. 使用 **內容屬性** 在右側面板上修改影像

   ·規模

   • position

   ·對齊，或

   ·其他屬性。

   ![內容屬性](images/lesson-15/content-properties.png)

## 處理多個主題

在比較主題、在主題之間複製和貼上或將內容從一個主題拖放到另一個主題時，拆分視圖很有幫助。

1. 開啟兩個或兩個以上相關主題。

2. 按一下一個檔案的「標題」頁籤以開啟上下文菜單。

3. 選擇 [!UICONTROL **拆分**]。

4. 選擇 **右**。

   ![拆分視圖](images/lesson-15/split-view.png)

## 更正排版錯誤

1. 找到包含錯誤的字詞或短語。

2. 按住 [!UICONTROL **Ctrl鍵**]。

3. 按一下錯誤上的輔助滑鼠按鈕。

4. 選擇正確的拼寫。

已在主題文本中更正錯誤。

## 下載主題PDF

用戶可能希望下載當前主題的PDF以標籤或與他人共用。

1. 按一下 [!UICONTROL **預覽**] 右上角。

2. 按一下 [!UICONTROL **PDF表徵圖**] 上面。 對話方塊隨即顯示。

   ![PDF導出](images/lesson-15/pdf-export.png)

3. 填寫任一資訊 **轉換名稱** 或 **DITA-OT命令行參數** 的子菜單。 請注意，如果所有欄位都留空，則仍會生成PDF。

4. 按一下&#x200B;[!UICONTROL **下載**]。PDF生成。

5. 使用可用表徵圖配置、下載或共用PDF主題。

## 在儲存庫或映射中查找主題

1. 開啟主題。

2. 按一下「標題」(Title)頁籤上的輔助滑鼠按鈕。

3. 選擇 **定位於**。

4. 選擇 **儲存庫** 或 **地圖** 跳到所需的主題位置。


## 版本主題

1. 更改主題。

2. 保存主題。

3. 按一下 **儲存庫** 表徵圖。

   ![儲存庫表徵圖](images/lesson-15/repository-icon.png)

4. 在對話框中，添加 **新版本的注釋**。

   ![「新版本」對話框](images/lesson-15/version-dialog.png)

5. 按一下「[!UICONTROL **儲存**]」。

版本號更新。

## 載入版本標籤

嘗試僅基於版本號跟蹤主題的狀態可能會很困難。 標籤使識別已經過多次修訂的主題的準確狀態變得更容易。

1. 選擇 **資料夾配置檔案**。

2. 在資料夾配置檔案中，配置XML編輯器。

   a.選擇螢幕左上角的編輯。

   b.在「XML內容版本標籤」下，添加新主題或使用現有主題。

   ![內容版本標籤](images/lesson-15/version-labels.png)

3. 選擇 [!UICONTROL **上載**]。

4. 選擇ReviewLabels.json或類似檔案。 有關如何建立此類檔案的詳細資訊將在另一視頻中介紹。

5. 按一下 [!UICONTROL **開啟**]。

6. 按一下 [!UICONTROL **保存**] 資料夾配置檔案螢幕左上角。

7. 按一下 [!UICONTROL **關閉**] 右上角。

現在已載入版本標籤。

## 分配版本標籤

1. 載入版本標籤。

2. 按一下 [!UICONTROL **用戶首選項**] 表徵圖。

   ![資料夾配置檔案](images/lesson-15/folder-profile-icon.png)

3. 選擇以前載入版本標籤的資料夾配置檔案。

4. 在「用戶首選項」對話框中，確保「基本路徑」引用與「資料夾配置檔案」已應用到的資訊相同的資訊。

   ![使用者偏好設定](images/lesson-15/user-preferences.png)

5. 按一下「[!UICONTROL **儲存**]」。

6. 版本主題。

7. 添加註釋並從下拉清單中選擇版本標籤。

   ![「新版本標籤」對話框](images/lesson-15/labels-dialog.png)

8. 按一下「[!UICONTROL **儲存**]」。

版本號更新。

## 查看版本歷史記錄和標籤

1. 從左側面板中，找到當前主題標題。

2. 按一下標題以開啟上下文菜單。

3. 選擇 [!UICONTROL **在資產UI中查看**]。

   ![資產UI](images/lesson-15/view-assets-ui.png)

   - 帶有標籤的版本歷史記錄顯示在左側。

   ![版本歷史記錄](images/lesson-15/version-history.png)


4. 按一下某個版本以訪問選項，如 **還原到此版本** 和 **預覽版本**。

## 建立新模板

主題和映射都有模板。 管理員可以訪問左面板中的「模板」。

1. 按一下 [!UICONTROL **模板**] 的下界。

2. 選擇「映射」或「主題」以開啟關聯的上下文菜單。

3. 按一下以添加新模板。

   ![新建主題模板](images/lesson-15/version-history.png)

4. 填充結果對話框中的欄位。

此時將出現殼模板，其中包含示例內容和示例結構。
