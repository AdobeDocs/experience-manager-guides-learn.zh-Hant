# AEM Guides擴充功能套件

AEM Guides擴充功能套件可讓您根據自己的需求自訂應用程式。 要在工具列上新增另一個按鈕嗎？ 註釋方塊中的新輸入欄位？ 內容選單中的另一個選項？ 現在您可以全部操作。

前往 [安裝指南](./integrating_customisations.md) 以瞭解如何使用此套裝

> **_注意：_** 此AEM Guides擴充功能套件受到AEM Guides version >= 4.3 （內部部署）和>= 2310 （雲端）支援
