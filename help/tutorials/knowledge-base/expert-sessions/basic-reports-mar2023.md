# 指南中的報AEM告

在組織設定中，您希望在推送文檔供最終用戶使用之前，驗證內容的整體完整性。 這種需要在多用戶和大規模環境中變得更加必要，在這些環境中，內容可以在不同的上下文中重新使用，因此驗證您的內容或文檔是必要的。 在結構化內容的上下文中，有一些通常需要的特定類型的報告。


## 會話概述

在此會話中，我們顯示了可供用戶使用的現成報告。 我們大致介紹了以下內容：
- 文檔（圖）和主題報告
- 元資料報告和批量元資料管理選項
- 內容重用報告
- 內容導入正確性
- 利用AEM Assets報告功能
