# 使用指南創AEM作內容 {#id221EB0P0E20}

指AEM南附帶三個現成編輯。 您可以使用它們來建立、編輯和發佈文檔。

1. **Web編輯器**:使用它可建立和編輯結構化文檔。
1. **映射編輯器**:使用它可建立和編輯DITA映射和書籤映射檔案。
1. **雙曲編輯器**:使用它生成條件輸出。

- **[DITAVAL編輯器](ditaval-editor.md)**
