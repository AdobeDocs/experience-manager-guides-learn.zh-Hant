# 建立和預覽主題 {#id21BQE0RJ042}

AEM參考線隨附強大的功能，可從資產UI建立DITA主題。 然後，您就可以從資產UI預覽主題。 「預覽」模式還提供許多強大功能，幫助您執行關鍵操作，如查看元資料、在Web編輯器中編輯主題、開啟源，以及查找基於Web的版本差異。
