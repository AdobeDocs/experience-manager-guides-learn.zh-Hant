# 建立和預覽主題 {#id21BQE0RJ042}

指AEM南附帶了一個功能強大的功能，可從資產用戶介面建立DITA主題。 然後，可以從「資產」UI預覽主題。 「預覽」模式還提供了許多功能強大的功能，可幫助您執行關鍵操作，如查看元資料、在Web編輯器中編輯主題、開啟源以及查找基於Web的版本差異。
