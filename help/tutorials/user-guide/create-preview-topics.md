# 建立和預覽主題 {#id21BQE0RJ042}

AEM Guides提供強大的功能，可從Assets UI建立DITA主題。 然後，您可以從Assets UI預覽主題。 「預覽」模式還提供許多強大的功能，可幫助您執行關鍵操作，例如檢視中繼資料、在Web編輯器中編輯主題、開啟原始碼以及尋找網頁型版本差異。
