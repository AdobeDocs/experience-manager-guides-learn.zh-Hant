# 瞭解輸出預設 {#understanding_output_settings}

指AEM南支援以PDF、HTML5和EPUB格式為FrameMaker文檔建立輸出。 使用這些輸出格式，可以配置各種輸出預設。 輸出預設表示要發佈內容的自定義輸出格式。

以下各節說明了支援的輸出格式的可用選項。
