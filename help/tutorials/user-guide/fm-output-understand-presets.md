# 了解輸出預設集 {#understanding_output_settings}

AEM參考線支援以PDF、HTML5和EPUB格式建立FrameMaker文檔的輸出。 您可以使用這些輸出格式來配置各種輸出預設集。 輸出預設集代表您要發佈內容的自訂輸出格式。

以下各節說明支援的輸出格式可用的選項。

**上層主題：**[&#x200B;生成FrameMaker文檔的輸出](fm-output-generatation.md)
