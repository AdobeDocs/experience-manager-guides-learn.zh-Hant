# 翻譯內容 {#id1645H700L67}

AEM指南提供強大的功能，可讓您將內容翻譯成多種語言。 AEM指南均支援人工和機器翻譯工作流程。

- **人類翻譯**  — 資產將發送給您的翻譯提供商，並由專業翻譯人員翻譯。 完成時，會傳回翻譯的資產，然後匯回AEM。

- **機器翻譯**  — 這是AEM中的預設服務，內容會使用機器翻譯即時立即翻譯。 現成可用的AEM提供連線至Microsoft Translator的功能。

   >[!NOTE]
   >
   > Microsoft Translator僅提供試用版許可。
