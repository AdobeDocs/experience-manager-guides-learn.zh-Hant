# Web編輯器中的其他功能 {#id2056B0B0YPF}

Web編輯器中還有一些其它有用的功能，您可以利用：

**檔案頁籤上的上下文菜單功能**

在Web編輯器中開啟檔案時，可以從上下文菜單執行各種操作。 您可能會看到不同的選項，具體取決於您是開啟媒體檔案、單個DITA檔案還是開啟多個檔案。

**媒體檔案**

在開啟的媒體檔案頁籤的上下文菜單中，可以獲得以下功能：

![](images/media-file-context-menu.png){width="300" align="left"}

**單個DITA檔案**

在開啟的檔案頁籤的上下文菜單中可獲得以下功能：

:   ![](images/single-file-context-menu.png){width="300" align="left"}

**多個檔案**

開啟多個檔案後，上下文菜單中會顯示更多選項：

![](images/multiple-files-context-menu.png){width="550" align="left"}

上下文菜單中的各種選項說明如下：

***保存***:您可以從以下選項中進行選擇：

- **保存**:要保存檔案而不建立新版本，請選擇 **保存**。 無論何時建立新主題，都會在DAM中建立主題的無版本工作副本。 保存文檔會更新DAM中文檔的工作副本。 對此版本執行簡單保存不會建立主題的新版本。 如果您的主題正在審閱，保存主題不會讓您的審閱者訪問更改的主題內容。

- **全部保存**:如果Web編輯器中開啟了多個文檔，則您還可以 **全部保存** 開啟的文檔。


***另存為新版本***

要建立檔案的新版本，請選擇 **另存為新版本**。 有關詳細資訊 **保存** 和 **另存為新版本**，請參閱 [瞭解Web編輯器功能](web-editor-features.md#)。

***複製***:您可以從以下選項中進行選擇：

- **複製UUID**:要將當前活動檔案的UUID複製到剪貼簿，請選擇 **複製\>複製UUID**。
- **複製路徑**:要將當前活動檔案的完整路徑複製到剪貼簿，請選擇 **複製\>複製路徑**。


***定位於***:您可以從以下選項中進行選擇：

- **地圖**:如果已開啟一個大型DITA映射並想在映射中查找檔案的準確位置，請選擇 **在\>映射中查找**。 選擇「在映射中查找」選項時，檔案\（從中調用該選項的位置\）將位於映射層次中並突出顯示。 要能夠使用此功能，必須在Web編輯器中開啟映射檔案。 如果「映射視圖」被隱藏，則調用此功能將顯示「映射視圖」，並且檔案在映射層次結構中突出顯示。

- **儲存庫**:與「在映射中查找」類似， **在\>儲存庫中查找** 顯示檔案在儲存庫\（或DAM\）中的位置。 「儲存庫視圖」(Repository View)已開啟，並且所選檔案在儲存庫中突出顯示。 如果檔案位於資料夾中，則會展開該資料夾以顯示選定檔案在儲存庫中的位置。


***添加到***:您可以從以下選項中進行選擇：

- **收藏夾**:要將所選檔案添加到收藏夾，請選擇 **添加到\>收藏夾**。 有關詳細資訊，請參閱 **收藏夾** 功能說明 [左面板](web-editor-features.md#id2051EA0M0HS) 的子菜單。



- **可重用內容**:要將所選檔案複製到可重用內容清單，請選擇 **添加到\>可重用內容**。 有關詳細資訊，請參閱 **可重用內容** 功能說明 [左面板](web-editor-features.md#id2051EA0M0HS) 的子菜單。




***屬性***

要查看所AEM選檔案的屬性頁，請選擇 **屬性**。

***拆分***:您可以從以下選項中進行選擇：

**上、下、左或右**

預設情況下，Web編輯器允許您一次查看一個主題。 有些實例可能希望同時查看兩個或多個主題。 拆分編輯器螢幕允許您同時查看多個主題。 例如，如果您有兩個主題 — A和B在編輯器中開啟。 按一下右鍵主題B並選擇 **拆分\>向上** 將編輯器窗口分為兩部分。 主題B顯示在上半部，主題A顯示在下半部。 同樣，也可以通過選擇 **拆分\>左** 或 **拆分\>右**。 Web編輯器的以下螢幕快照顯示水準和垂直拆分的主題。 在每個拆分中，可以有不同的視圖。 例如，在以下螢幕截圖中，螢幕1處於「源」視圖模式，螢幕2在「作者」模式下開啟了兩個文檔，螢幕3處於「預覽」模式。 通過拖動檔案頁籤並將其放到要放置文檔的螢幕上，可以將文檔從一個螢幕移動到另一個螢幕。 同樣，您也可以根據首選項拖動和移動檔案頁籤來重新排序檔案頁籤。

![](images/split-editor.png){width="800" align="left"}

***快速生成***

為所選檔案生成輸出。 只能為作為輸出預設一部分的檔案生成輸出。 有關詳細資訊，請參閱 [基於文章的Web編輯器發佈](web-editor-article-publishing.md#id218CK0U019I)。

***關閉***:您可以從以下選項中進行選擇：

**關閉**。 **關閉其他**&#x200B;或 **全部關閉**

如果要關閉從中調用上下文菜單的檔案，請選擇 **關閉\>關閉**。 使用 **關閉\>關閉其他** 關閉除當前活動檔案之外的所有其他開啟的檔案。 要關閉所有開啟的檔案，請選擇 **關閉\>全部關閉** 選項，也可以選擇關閉Web編輯器。 如果會話中有任何未保存的檔案，則系統會提示您保存這些檔案。

**檔案關閉和保存方案**

當嘗試使用 **關閉** 的子菜單。 **關閉** 選項菜單中的輔助AEM線提示您保存編輯並解鎖鎖定的檔案。

提示基於管理員選擇的以下配置：

- **在關閉時要求籤入：** 在關閉編輯器時，您可以選擇簽入檔案\（您已簽出該檔案\）。
- **在關閉時請求新版本**:在關閉編輯器時，您可以選擇將檔案\（已編輯的\）另存為新版本。

您的檔案保存體驗將取決於以下三種情形，其中您有：

- 未對內容進行任何更改。
- 已編輯內容並保存更改。
- 已編輯內容，但未保存更改。

根據檔案是否已鎖定/解鎖，以及是否保存或未保存更改，您可能會看到以下選項：

- **解鎖和關閉**:檔案上的鎖被解除，檔案被關閉。

   ![](images/file-close-unlock-file.png){width="400" align="left"}

- **另存為新版本**:這將保存您在內容中所做的更改並建立檔案的新版本。 您也可以為新保存的版本添加標籤和注釋。 有關保存新版本的詳細資訊，請參見 [另存為新版本](web-editor-features.md#save-as-new-version-id209ME400GXA)。

- **解鎖檔案**:如果選擇解鎖檔案，它將釋放檔案上的鎖，更改將保存在檔案的當前版本中。

>[!NOTE]
>
> 如果取消選擇解鎖檔案的選項，則還可以選擇關閉檔案而不保存更改。

例如，以下螢幕快照中顯示了其中一個提示：

![](images/file-close-save-changes-unlock.png){width="400" align="left"}

**斷開引用的視覺提示**

- 如果您的主題包含斷開的交叉引用或內容引用，則它們將以紅色文本顯示。

**智慧複製貼上**

- 您可以輕鬆地在主題內和主題間複製貼上內容。 源元素結構被保持在目的地。 此外，如果複製的內容包含內容引用，則甚至會複製這些內容。

**記住上次瀏覽的位置**

- Web編輯器提供智慧檔案瀏覽對話框。 編輯器在插入引用或內容時記住上次使用的位置。 首次調用檔案瀏覽對話框\（通過「插入引用」或「插入重用內容」\）時，您將被帶到當前文檔的保存位置。 在同一會話中，如果嘗試插入另一個參照，則檔案瀏覽對話框將自動導航到插入最後一個參照的位置。

>[!NOTE]
>
> 對於影像、音頻或視頻檔案，檔案瀏覽對話框預設為檔案的位置，而不是上次使用的位置。

**支援基於文章的發佈**

- 從Web編輯器中，可以為一個或多個主題或整個DITA映射生成輸出。 您需要為DITA映射建立輸出預設，然後可以輕鬆地為一個或多個主題生成輸出。 如果更新了映射中的幾個主題，您也只能從Web編輯器中為這些主題生成輸出。 有關詳細資訊，請參閱 [基於文章的Web編輯器發佈](web-editor-article-publishing.md#id218CK0U019I)。

**支援Markdown文檔**

- Web編輯器允許您將標籤文檔\(.md\)與DITA文檔一起使用。 您可以輕鬆地在Web編輯器中建立和預覽Markdown文檔，還可以通過DITA映射編輯器將其添加到映射檔案中。 有關詳細資訊，請參閱 [從Web編輯器建立標籤文檔](web-editor-markdown-topic.md#)。

**支援DITA術語主題**

- Web編輯器支援通過添加可插入的DITA術語 `term` 或 `abbreviated-form` 元素。

**插入MathML方程**

- 參AEM考線為您提供了通過與整合插入MathML方程的現成支援 [數學類型Web](https://docs.wiris.com/en/mathtype/mathtype_web/intro) 的子菜單。 要插入MathML公式，請按一下 **插入元素** 表徵圖，然後鍵入mathml。 從清單中選擇mathml元素時，將顯示「插入MathML」對話框：

![](images/insert-mathml-equation.png){width="550" align="left"}

使用MathML等式工具建立公式，然後按一下「插入」將其添加到文檔。 公式插入有淺灰色背景，如下所示：

![](images/sample-mathml-equation.PNG){width="400" align="left"}

您可以隨時通過按一下右鍵現有公式並選擇 **編輯MathML** 的子菜單。

**插入腳注**

- 使用 `fn` 的子菜單。 在創作模式中，腳注值與內容一起顯示。 但是，當您切換「預覽」模式或發佈文檔時，腳注將出現在主題的末尾。

**更名或替換元素**

- Web編輯器將在主題頂部顯示元素的breadcrumb。 如果要用另一個元素交換或替換元素，則可以從breadcrumb的上下文菜單執行此操作。 例如，您可以 `p` 元素 `note` 或上下文中的任何其他有效元素。

![](images/rename-element.png){width="400" align="left"}

在breadcrumb上，按一下右鍵要替換的元素名稱，然後從上下文菜單中選擇「更名元素」。 「更名元素」(Rename Element)對話框顯示當前位置允許的所有有效元素。 從「更名元素」(Rename Element)對話框中，選擇要使用的元素。 原始元素將替換為新元素。

除了breadcrumb的上下文菜單外，還可以從其他位置訪問「更名元素」對話框：

- 按一下breadcrumb上的元素名稱以選擇元素的內容，然後按一下右鍵所選內容以開啟上下文菜單。

- 啟用「標籤」視圖，按一下任何元素的開啟標籤，然後按一下右鍵選定內容以開啟上下文菜單。

- 通過調用「大綱」面板中某個元素的「選項」菜單，可以訪問「更名元素」對話框。


**取消對元素的包絡**

- 展開元素允許您從選定文本中移除元素標籤並將其與其父元素合併。 例如，如果 `p` 元素 `note` 元素，可以取消包裝 `p` 元素，將文本直接合併到 `note` 的子菜單。 在主題的breadcrumb的上下文菜單中，「取消換行元素」選項可用。 要取消對元素的包裝，請從主題的breadcrumb中選擇元素，然後按一下右鍵元素以開啟上下文菜單，最後選擇「取消對元素」以刪除元素並將元素的文本與其父元素合併。

**保留換行符和縮進**

- 在「作者」、「源」或「預覽」模式以及最終發佈的輸出中，都支援包含換行符和空格的DITA元素並按照其定義進行渲染。 以下螢幕快照顯示 `msgblock` 元素，其中保留了換行符和空格\（縮進\）:

![](images/new-line-support_cs.png){width="500" align="left"}

**自動生成元素ID**

- 可以自動為DITA主題中的元素生成ID。 這些ID在DITA主題中是唯一的。 例如，如果為段落元素生成ID，則ID將為p\_1、p2、p\_3等。 可以選擇多個元素，並為每個選定元素生成ID。

執行以下操作可自動為一個或多個元素生成ID:

1. 在Web編輯器中開啟主題。
1. 選擇要分配ID的內容。
1. 按一下右鍵並選擇 **從上下文菜單生成ID。**

   或者，可以在breadcrumb中按一下右鍵並選擇 **生成ID**。
