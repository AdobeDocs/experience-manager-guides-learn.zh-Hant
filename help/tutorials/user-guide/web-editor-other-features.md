# Web編輯器中的其他功能 {#id2056B0B0YPF}

網頁編輯器中還有其他一些實用功能可供使用：

**檔案索引標籤上的內容功能表功能**

在Web編輯器中開啟檔案時，可以從上下文菜單執行各種操作。 視您開啟的媒體檔案、單一DITA檔案或多個檔案而定，您可能會看到不同的選項。

**媒體檔案**

您可在已開啟媒體檔案索引標籤的內容功能表中取得下列功能：

![](images/media-file-context-menu.png){width="300" align="left"}

**單一DITA檔案**

在已開啟檔案索引標籤的內容功能表中，您會取得下列函式：

:   ![](images/single-file-context-menu.png){width="300" align="left"}

**多個檔案**

開啟多個檔案後，上下文菜單中會顯示更多選項：

![](images/multiple-files-context-menu.png){width="550" align="left"}

內容功能表中的各種選項說明如下：

***儲存***:您可以選擇下列選項：

- **儲存**:要保存檔案而不建立新版本，請選擇 **儲存**. 每當您建立新主題時，就會在DAM中建立主題的無版本工作副本。 儲存檔案會更新DAM中檔案的工作副本。 對此版本執行簡單儲存不會建立主題的新版本。 如果正在審核您的主題，保存主題不會讓審核者訪問已更改的主題內容。

- **全部儲存**:如果Web編輯器中開啟了多個文檔，則您還可以選擇 **全部儲存** 已開啟文檔。


***另存為新版本***

要建立新版本的檔案，請選擇 **另存為新版本**. 如需詳細資訊，請參閱 **儲存** 和 **另存為新版本**，請參閱 [了解Web編輯器功能](web-editor-features.md#).

***複製***:您可以選擇下列選項：

- **複製UUID**:若要將目前作用中檔案的UUID複製到剪貼簿，請選取 **複製\>複製UUID**.
- **複製路徑**:要將當前活動檔案的完整路徑複製到剪貼簿，請選擇 **複製\>複製路徑**.


***在中查找***:您可以選擇下列選項：

- **地圖**:如果您已開啟大型DITA映射，並且想在映射中查找檔案的確切位置，請選擇 **在「\>對應」中找到**. 選擇「在地圖中查找」選項時，將在地圖層次結構中查找並突出顯示檔案\（從調用該選項的位置\）。 若要使用此功能，您必須在網頁編輯器中開啟對應檔案。 如果「地圖檢視」隱藏，則叫用此功能會顯示「地圖檢視」，且在地圖階層中會反白顯示檔案。

- **存放庫**:類似於「在地圖中找出」， **在\>儲存庫中找到** 顯示檔案在存放庫\（或DAM\）中的位置。 「儲存庫視圖」(Repository View)開啟，選定檔案在儲存庫中突出顯示。 如果檔案位於資料夾中，則會展開該資料夾以顯示所選檔案在儲存庫中的位置。


***添加到***:您可以選擇下列選項：

- **我的最愛**:要將所選檔案添加到收藏夾，請選擇 **添加到\>收藏夾**. 如需詳細資訊，請參閱 **我的最愛** 功能說明，請參閱 [左面板](web-editor-features.md#id2051EA0M0HS) 區段。



- **可重複使用的內容**:要將所選檔案複製到可重複使用的內容清單，請選擇 **添加到\>可重複使用的內容**. 如需詳細資訊，請參閱 **可重複使用的內容** 功能說明，請參閱 [左面板](web-editor-features.md#id2051EA0M0HS) 區段。




***屬性***

若要查看所選檔案的AEM屬性頁面，請選取 **屬性**.

***分割***:您可以選擇下列選項：

**上、下、左或右**

依預設，網頁編輯器可讓您一次檢視一個主題。 有時您想要同時看到兩個或多個主題。 分割編輯器的畫面可讓您同時檢視多個主題。 例如，如果您有兩個主題 — 在編輯器中開啟了A和B。 按一下右鍵主題B並選擇 **拆分\>向上** 將編輯器視窗分為兩部分。 主題B顯示在上半部，主題A顯示在下半部。 同樣地，您也可以選取 **分割\>左** 或 **拆分\>右**. 以下Web編輯器螢幕截圖顯示水準和垂直拆分的主題。 在每個分割中，您可以有不同的檢視。 例如，在以下螢幕截圖中，螢幕1處於源視圖模式，螢幕2在作者模式下開啟了兩個文檔，螢幕3處於預覽模式。 通過將檔案頁簽拖放到要放置文檔的螢幕上，可將文檔從一個螢幕移動到另一個螢幕。 同樣地，您也可以根據您的偏好設定來拖放和移動檔案標籤，以重新排序檔案標籤。

![](images/split-editor.png){width="800" align="left"}

***快速產生***

為所選檔案生成輸出。 只能為輸出預設集的一部分的檔案生成輸出。 如需詳細資訊，請參閱 [從網頁編輯器發佈以文章為基礎](web-editor-article-publishing.md#id218CK0U019I).

***關閉***:您可以選擇下列選項：

**關閉**, **關閉其他**，或 **全部關閉**

如果要關閉從中調用上下文菜單的檔案，請選擇 **關閉\>關閉**. 使用 **關閉\>關閉其他** 以關閉除當前活動檔案之外的所有其他開啟的檔案。 要關閉所有開啟的檔案，請選擇 **關閉\>全部關閉** 選項，或者您也可以選擇關閉Web編輯器。 如果您的工作階段中有任何未儲存的檔案，則系統會提示您儲存這些檔案。

**檔案關閉和保存方案**

當您嘗試關閉在網頁編輯器中開啟的檔案時，請使用 **關閉** 按鈕，或 **關閉** 選項，AEM參考線會提示您儲存編輯項目並解除鎖定檔案。

提示是根據管理員選擇的以下配置：

- **在關閉時要求籤入：** 在關閉編輯器時，您可以選擇簽入檔案\（您已簽出檔案\）。
- **關閉時請求新版本**:您可以選擇在關閉編輯器時將檔案\（您已編輯檔案\）儲存為新版本。

您的檔案儲存體驗取決於下列三種情況，其中您有：

- 未對內容進行任何變更。
- 編輯內容並儲存變更。
- 已編輯內容，但未儲存變更。

根據檔案已鎖定/解除鎖定，且已儲存或未儲存的變更，您可能會看到下列選項：

- **解鎖和關閉**:檔案的鎖已釋放，檔案將關閉。

   ![](images/file-close-unlock-file.png){width="400" align="left"}

- **另存為新版本**:這會儲存您在內容中所做的變更，並建立新版本的檔案。 您也可以為新儲存的版本新增標籤和註解。 如需儲存新版本的詳細資訊，請參閱 [另存為新版本](web-editor-features.md#save-as-new-version-id209ME400GXA).

- **解除鎖定檔案**:如果選擇解除鎖定檔案，該檔案將釋放對檔案的鎖定，且更改將保存在檔案的當前版本中。

>[!NOTE]
>
> 如果您取消選取解除鎖定檔案的選項，您也會獲得關閉檔案而不儲存變更的選項。

例如，以下螢幕截圖中顯示了其中一個提示：

![](images/file-close-save-changes-unlock.png){width="400" align="left"}

**損壞引用的視覺提示**

- 如果您的主題包含中斷的交叉參照或內容參考，則會以紅色文字顯示。

**智慧複製貼上**

- 您可以在主題內和主題之間輕鬆複製貼上內容。 源元素結構在目標處保持。 此外，如果複製的內容包含內容參考，則即使是複製的內容亦然。

**記住上次瀏覽的位置**

- Web編輯器提供智慧檔案瀏覽對話框。 編輯器會在插入參考或內容時記住上次使用的位置。 首次調用檔案瀏覽對話框時（通過「插入引用」或「插入重複使用內容」），將轉到當前文檔的保存位置。 在同一會話中，如果嘗試插入其他引用，則檔案瀏覽對話框將自動導航到插入最後一個引用的位置。

>[!NOTE]
>
> 若是影像、音訊或視訊檔案，檔案瀏覽對話方塊預設為檔案的位置，而非上次使用的位置。

**支援文章發佈**

- 從Web編輯器，可以為一個或多個主題或整個DITA映射生成輸出。 您需要為DITA映射建立輸出預設集，然後可以輕鬆為一個或多個主題生成輸出。 如果您已更新地圖中的幾個主題，您也只能從網頁編輯器為這些主題產生輸出。 如需詳細資訊，請參閱 [從網頁編輯器發佈以文章為基礎](web-editor-article-publishing.md#id218CK0U019I).

**支援Markdown檔案**

- Web編輯器可讓您使用Markdown檔案\(.md\)和DITA檔案。 您可以在網頁編輯器中輕鬆製作和預覽Markdown檔案，並透過DITA地圖編輯器將其新增至您的地圖檔案。 如需詳細資訊，請參閱 [從網頁編輯器製作Markdown檔案](web-editor-markdown-topic.md#).

**支援DITA字彙表術語主題**

- Web編輯器支援DITA字彙表術語，您可以借由新增 `term` 或 `abbreviated-form` 元素。

**插入MathML方程**

- AEM參考線可讓您透過與 [MathType Web](https://docs.wiris.com/en/mathtype/mathtype_web/intro) 應用程式。 若要插入MathML方程式，請按一下 **插入元素** 圖示並輸入mathml。 從清單中選擇mathml元素時，將顯示「插入MathML」對話框：

![](images/insert-mathml-equation.png){width="550" align="left"}

使用MathML方程式工具建立方程式，然後按一下「插入」(Insert)將其添加到文檔中。 方程式會插入淺灰色背景，如下所示：

![](images/sample-mathml-equation.PNG){width="400" align="left"}

您隨時都可以按一下滑鼠右鍵現有方程式並選取「 」，以更新方程式 **編輯MathML** 從上下文菜單。

**插入腳注**

- 使用 `fn` 元素。 在編寫模式中，腳注值會與內容內嵌顯示。 但是，當您切換「預覽」模式或發佈文檔時，腳注將出現在主題的末尾。

**重新命名或取代元素**

- 網頁編輯器會在主題頂端顯示元素的階層連結。 如果您想要交換元素或將元素取代為其他元素，則可從階層連結的內容功能表執行此操作。 例如，您可以交換 `p` 元素 `note` 或內容中的任何其他有效元素。

![](images/rename-element.png){width="400" align="left"}

在階層連結上，以滑鼠右鍵按一下您要取代的元素名稱，然後從內容功能表選取「重新命名元素」 。 「更名元素」對話框顯示當前位置允許的所有有效元素。 從「更名元素」對話方塊中，選取您要使用的元素。 原始元素會取代為新元素。

除了階層連結的內容功能表外，您也可以從其他位置存取「重新命名元素」對話方塊：

- 按一下階層連結上的元素名稱，以選取元素的內容，然後按一下滑鼠右鍵選取的內容以開啟內容功能表。

- 啟用「標籤」檢視，按一下任何元素的開頭標籤，然後以滑鼠右鍵按一下選取的內容以開啟內容功能表。

- 通過調用「大綱」面板中某個元素的「選項」菜單，可以訪問「更名元素」對話框。


**取消繞排元素**

- 展開元素可讓您從選取的文字中移除元素標籤，並將其與其父元素合併。 例如，如果您有 `p` 元素內 `note` 元素，您可以取消包裝 `p` 元素，直接在 `note` 元素。 主題階層連結的內容功能表中提供「取消換行元素」選項。 若要取消繞排元素，請從主題的階層連結中選取元素，然後以滑鼠右鍵按一下元素以開啟內容功能表，最後選取「取消繞排元素」以移除元素，並合併元素的文字與其父元素。

**保留斷行和縮進**

- 包含分行符號和空格的DITA元素在製作、來源或預覽模式以及最終發佈輸出中均支援並根據其定義呈現。 以下螢幕擷圖顯示 `msgblock` 元素，其中已保留分行和空格\（縮排\）:

![](images/new-line-support_cs.png){width="500" align="left"}

**自動產生元素ID**

- 您可以自動為DITA主題中的元素產生ID。 這些ID在DITA主題中是唯一的。 例如，如果您為段落元素產生ID，則ID會是p\_1、p2、p\_3等。 您可以選取多個元素，並為每個選取的元素產生ID。

執行下列操作以自動為一或多個元素產生ID:

1. 在網頁編輯器中開啟主題。
1. 選取要指派ID的內容。
1. 按一下滑鼠右鍵並選取 **從內容功能表產生ID。**

   或者，您也可以在階層連結中按一下滑鼠右鍵並選取 **產生ID**.
